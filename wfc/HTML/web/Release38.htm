Release 38

Editorial

This release is maintenance release. It fixes a ton of bugs and adds very little new capability. Most of the work has been focused on the XML classes. I've also added the ability to statically link with WFC.

What's New

CSharedMemoryObject - Let's you create shared memory segments.

Bug Fixes/Enhancements

CBitArray - Got rid of floating point code. Fixed numerous nit noid bugs. Changed the spelling of the Complement() method.

CExtensibleMarkupLanguageDocument - I've added write options and parsing options. This allows you to customize the output as well as slop up the parser.

CExtensibleMarkupLanguageElement - Multiple bugs fixed here.

  • CDATA sections not properly terminated.
  • CDATA sections not properly parsing ']' characters. Found and fixed by ME!
  • Attributes delimited by single quote's are properly handled
  • Automatic addition of entity tags
  • Missing XML declarations are now handled correctly
