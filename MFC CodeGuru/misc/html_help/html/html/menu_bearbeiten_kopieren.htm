Der Befehl Kopieren (Menü Bearbeiten)

Verwenden Sie diesen Befehl, um markierte Daten in die Zwischenablage zu kopieren. Der Befehl kann nicht ausgewählt werden, wenn momentan keine Daten markiert sind.

Das Kopieren von Daten in die Zwischenablage ersetzt die Daten, die sich zuvor darin befanden.

Abkürzungen

Symbolleiste:

Tastatur: STRG+C
