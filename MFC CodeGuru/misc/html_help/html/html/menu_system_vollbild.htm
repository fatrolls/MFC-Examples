Der Befehl Vollbild (Systemmenü)

Verwenden Sie diesen Befehl zur Vergrößerung des aktiven Fensters, so daß es den gesamten verfügbaren Platz einnimmt.

Abkürzungen

Maus: Klicken Sie auf die Schaltfläche für Maximieren in der Titelleiste, oder doppelklicken Sie auf die Titelleiste.

Tastatur: STRG+F10 vergrößert ein Dokumentfenster.
