Der Befehl Symbolleiste (Menü Ansicht)

Verwenden Sie diesen Befehl zum Ein- oder Ausblenden der Symbolleiste. Die Symbolleiste enthält einige Schaltflächen der gebräuchlichsten Befehle von SabaBrandenburg, wie zum Beispiel Datei Öffnen. Wenn die Symbolleiste angezeigt wird, erscheint ein Häkchen neben diesem Menüeintrag.
