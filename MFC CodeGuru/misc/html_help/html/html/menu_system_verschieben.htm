Der Befehl Verschieben (Systemmenü)

Verwenden Sie diesen Befehl, um einen Vierfachpfeil anzeigen zu lassen, mit dem Sie das aktive Fenster oder Dialogfeld durch Drücken der RICHTUNGSTASTEN verschieben können.

Hinweis: Dieser Befehl ist nicht verfügbar, wenn das Fenster als Vollbild dargestellt wird.

Abkürzung

Tastatur: STRG+F7
