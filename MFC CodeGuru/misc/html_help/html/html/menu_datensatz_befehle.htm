Befehle des Menüs Datensatz

Das Menü Datensatz enthält die folgenden Befehle:

Erster Datensatz Zeigt den ersten Datensatz innerhalb der aktuellen Auswahl an
Vorheriger Datensatz Zeigt den vorherigen Datensatz innerhalb der aktuellen Auswahl an
Nächster Datensatz Zeigt den nächsten Datensatz innerhalb der aktuellen Auswahl an
Letzter Datensatz Zeigt den letzten Datensatz innerhalb der aktuellen Auswahl an
