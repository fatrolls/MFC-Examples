Befehle des Menüs Bearbeiten

Das Menü Bearbeiten enthält folgende Befehle:

Rückgängig Macht die letzte Bearbeitungsoperation rückgängig
Ausschneiden Löscht Daten aus dem Dokument und überträgt sie in die Zwischenablage
Kopieren Kopiert Daten aus dem Dokument in die Zwischenablage
Einfügen Fügt Daten aus der Zwischenablage in das Dokument ein
Verknüpfung einfügen Fügt eine Verknüpfung zu Daten einer anderen Anwendung aus der Zwischenablage ein
Neues Objekt einfügen Fügt und bettet ein Objekt, wie zum Beispiel eine Tabelle oder eine Gleichung, ein
Verknüpfungen Listet Verknüpfungen zu eingebetteten Dokumenten auf und bearbeitet diese Verknüpfungen
