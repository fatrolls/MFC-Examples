Der Befehl Teilen (Menü Fenster)

Verwenden Sie diesen Befehl, um das aktive Fenster in Ausschnitte zu teilen. Daraufhin können Sie die Maus oder die RICHTUNGSTASTEN zur Verschiebung der Fensterteiler benutzen. Sobald Sie damit fertig sind, betätigen Sie die Maustaste oder die EINGABETASTE, um die Fensterteiler in ihrer neuen Position zu belassen. Durch Drücken von ESC bleiben die Fensterteiler in ihrer ursprünglichen Position. << In a single document interface application, this command will appear on the View menu. >>
